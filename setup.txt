# PHP 課程所需軟體安裝說明


1.  安裝 XAMPP

    下載並以系統管理員身份執行安裝 XAMPP。下載網址如下:
    https://www.apachefriends.org/zh_tw/index.html

    <Note>
    安裝完成後，請試著啟動 Apache 與 MySQL，
    確認是否順利執行。


2.  安裝 Visual Studio Code
    https://code.visualstudio.com/


3.  在 Visual Studio Code 安裝下列套件
    <Hint>
    點按 VS Code 左邊側邊工具的第五個(方形那個)圖示，
    輸入關鍵字，例如: php

    Live Server (ritwickdey.liveserver)
    PHP IntelliSense (felixfbecker.php-intellisense)
    PHP Server (brapifra.phpserver)
    

4.  將 php 課程資料夾複製到 c:\xampp\htdocs
    完成後的資料夾路徑:
    c:\xampp\htdocs\php

    <Note>
    如果是 MAMP，預設資料夾是:
    /Applications/MAMP/htdocs


5.  編輯程式: 以 code 開啟 php 資料夾
    執行程式: http://localhost/php
